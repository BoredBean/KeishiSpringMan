◆モデルについての解説

・※印の付いたモーフはグループモーフ用のモーフです。個々では使用しないでください。

・「 萃夢想、緋想天版」には、非想天則に登場する別カラーへの色替えモーフが付いてい
　ますが、一部のMMEエフェクト上ではきちんと機能しないようです。今の所対策方法が分
　からないのでその場合は諦めてください。

・本モデルは特に扇子や幽霊が原因で非常に重たいモデルになってしまっています（申し訳
　ありません）。そのため、扇子や幽霊を取り除いたモデルも付属させました。もし本モデ
　ルが重いという場合には、それら取り除いたモデルに必要なパーツだけを取り込んでご使
　用ください。「インポート用ファイル」フォルダにある※印の付いたpmxファイルはその
　ためのものです。取り込む手順は以下の通りです。
　
　①まずPMDエディターに「幽霊、扇子無」モデルを読み込みます。
　②次に「ファイル」→「インポート」で必要なpmxファイル（ファイル名に※が付いたもの）
　　を取り込みます。その際、同名ボーンのマージには必ずチェックを入れておいてください。
　③最後に念のためファイル名を替えてこのファイルを保存してください。

・幽霊、特大扇子、蝶は、そぼろ様製作のMMEエフェクト「AutoLuminous」に対応しており、
　使用することで光らせることができます。

・本モデルで使用しているスフィアマップは、スフィアマッＰ様製作の「hair2-b.bmp」
　「imas-r.bmp」を加工して使用させていただきました。
