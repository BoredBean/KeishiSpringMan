同梱モデルについて

東方三月精コミックス(比良坂先生画)を参考にさせていただきました…が
あんまり似ませんでした orz...
コミックスの霊夢の雰囲気、直球ど真ん中なんだけどなぁ
は置いておいて、以下留意点など

デフォルト表情について

    * V1.2aから V1.3でデフォルト表情をまた変更しました。モーフの関係で
      元のデフォルト表情にもどしました。

    * V1.1から V1.2でデフォルト表情を少し変えています。V1.1までの表情は
      縁側でお茶を飲んでいるような呑気な感じにしていましたが、踊らせる
      と「心ここにあらず」な感じになってしまうので、若干「キリッ」っと
      させました。
      従来の表情がいいという方は、その他カテゴリに「のんき」があります
      ので 0フレームにそれを登録してください。

モデル自体について

    * V1.2aでは接合部のウェイトにミスがあり、完全にミスを修正しきれて
      ませんでした。今回の修正で大分治ったかと思います。
      

    * 頭と首の接合部のポリ割および法線がおかしかったのを修正しています。
      指摘頂いておきながら長い間の放置申し訳ありませんでした。
    * ひじに「肘捻」を追加しました。それに伴い、肘を肘捻で捻じった場合
      は、袖は追従しません(それが本当の挙動だと思います)。
      従来の「ひじ」を捩じっているモーションももちろん流用できます。
      ※ 私のコダワリなので、どーでもいいのですが、モーション流用を前提
         としない場合はできるだけ、「肘捻」を使ってもらうとうれしいかも。
      
    * PMDEの検証で警告がギリギリ出てしまうくらいには重たいと思います。
      (出ないようにがんばったけど、だめでした)
    * 身長は、下駄はいた状態(とれないけど)で、ミクVer2さんより少し大きい
      くらいです。ルシールさん魔理沙となるべくバランスが取れるように
      してみました。
      1〜2%程度のスケール変更なら違和感はないと思いますので、もし必要で
      あればサイズ調整してください。
      ルシールさん魔理沙に合わせたので文句は彼まで(嘘
    * コメで「頭が大きい」とのご指摘を受けますが、逆です。他のモデルが小
      さすぎます(表現的にありなので否定する訳ではありません)。
      幻想郷の時代背景や日本人の一般体型および年齢より、初期の段階から
      身体のバランスはとっています。無論、対肩幅との比較では大きいですが
      2次絵自体がそういうものなので、3Dにしたときに違和感ない落としどころ
      で設計しています。

物理演算について

    * V1.1より、くろら様のPMDEプラグイン、PMDE用スカート自動設定プラグイン
      (sm14078798)を使用させていただきました。今までの苦労はなんだった
      んだw
      くろら様、素敵プラグインありがとうございます。
      (それでも、モーションを選ぶモデルだと思っています)
    * 袖に関しては、モデリングし直しと剛体構造の変更(V1.2)を行いましたが
      黒ポリ(反転ポリが表面に突き抜けるモノ)の発生を抑えられなかったため
      従来の構造に戻しました。
      ただ、ボーンの構成を変更したのと、初期で後45度に傾けたため、若干の
      違いはあります。
      また、V1.1までは、袖の破綻を防ぐため、あえて衝突判定をしなかった剛体
      がありますが、V1.2ではそれを有効にしています。
    
モーフについて

    * V1.3にて、眉のデフォルト形状を変更のうえ、リップに 3種、涙 1種を追加し
      ています。また、眉が顔に埋もれてしまうのを避けるために眉、前を追加しま
      した
    * 「涙だー」は「なごみ」をベースに調整していますが、各モーフの組み合わせ
      調整で他のものにも使えるかと思います。
    
    * V1.2で、眉の「下２」に加えて「下２左のみ」と「下２右のみ」を追加しました。
      「あぁん」(Hなほうじゃなくて、さげすむほう)みたいな表現のときに左右の眉
      のどちらかだけ下げたい場合に使ってください。
      あと、その他カテゴリから眉カテゴリに移動しています。
    * V1.2で、「ほっぺの〜を消す」モーフを加えました。ほっぺのあの表現も一種の
      比良坂さんキャラの特徴ではあるとは思いますが、一応用意しました。
    * 陰陽玉表示を含めて、モーフ(表情)は 54種類用意してあります。
      添付の reimuMorph.pngに顔のモーフのみ(49個)まとめておきました。
      表情を付ける際の参考にしてください。
      (表情名と実際の表情の対応についてはお手数ですが実際にいじってみ
      てメモとかしてください)
   
    * べろ出し等は、「い」「舌出し」「舌した」などの順番で自然になるよ
      うにしてます。
    
    * ○目などを使う場合は、瞳を動かして白目外に追い出してください。
   
    * 眉の下2は、下だけでは足りない場合に使ってください。
      キリッっとした目を表現するには目尻を上げて、眉を下ろした後、追加で
      下2を使って調整するといい結果がでるかもです。
   
    * のっぺらぼうの表情がありますが、これと青ざめを合わせることで
      マンガで良くあるような、内なる怒りを表現できると思います。
   
    * 歯は時として非常に邪魔になるときがあります。消せるようになって
      ますので動画の雰囲気に合わせて歯消しをつかってください。
   
    * このモデルに限ったことではありませんが、表情は組み合わせによっては
      非常に相性が悪いものがあります。
      たとえば「はぅ」などは付けた後、自動まばたき機能を使う場合、確認
      することをお勧めします。
   
陰陽玉について

    * 表情-その他に陰陽玉表示があります。
    * 陰陽玉は、MMDエンジンによって自動的に回転します。向かって頭右上にある
      捻じりボーン(○のなかに×があるやつ)を適当に弄ってみてください。
    * MMDエンジンですので FPSに依存します。
