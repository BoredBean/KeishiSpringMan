◆モデルについて

・傘の有無、スキマAの有無により４種類のモデルデータを入れてありますので、状況に
　応じて使い分けてください。


・モーフの中には※印の付いたモーフがありますがこれはグループモーフ用のモーフです。
　原則として個々では使用しないでください。


・色を替えるモーフについて、一部のMMEエフェクト上ではきちんと機能しないようです。
　今の所対処方法が分からないのでその場合は諦めてください。


・「靴消去モーフ」等、消去系のモーフは対象物が透明になるだけで実際に無くなるわけ
　ではありません。そのためMMEエフェクトの種類によっては、使用時に完全に透明にする
　ことが出来ず、かすかに残ってしまうことがあります。もし完全に消去したい場合には
　PMDEditorの材質削除を使用してください。


・スキマについて

　スキマAとスキマBの2種類があります。
　スキマAはスキマの中に紫が入ることが出来ます。（01.jpgのようなことができます）
　ただし透過機能を使用しているため、MMEエフェクトによっては使用時に完全に消えない
　という不具合が生じます。
　一方、スキマBはMMEエフェクト使用時の不具合は発生しませんが、中に入ることは出来ま
　せん。（公開用動画に使用したのはスキマBの方です）

○スキマAの使用方法
　「隙間A有り」と書かれたモデルをMMDに読み込んで使用してください。

○スキマBの使用方法
　「隙間無し」と書かれたモデルと「隙間B（背景）」、「隙間B（目A）」〜「隙間B（目F）」、
　「隙間B（枠）」をそれぞれMMDに読み込んで使用してください。スキマBはいわゆる紳士枠と
　呼ばれる方法を使用しています。以前作成したお空モデルの宇宙マントと同じ構造ですので
　詳しくはお空モデルのreadme.txtを参考にしてください。
