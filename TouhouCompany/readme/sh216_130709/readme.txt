⊂！！使用上の注意点！！⊃

モデル自体について ** ！！注意！！ **

    * 当モデルは、MMD向けのモデルであり、フリー素材でも何でもありません。
      MMDおよび、そこから派生した関連ツールでの使用は構いませんが、
      Blender、C4D、その他、直接関係のないツールに取り込んでの利用は認めま
      せん(改造などで一時的に取り込むのは構いません)。Unityもしかりです。
      
    * 普通にご利用していただいている方々には不快に感じる文面になってしまっ
      ていると思いますが、ご了承ください。

関節について

    * 基本的に球体関節ですので、人間ではムリな方向への曲げも可能ですが、
      その場合の動作は保証しません(一部のダンスモーションでもこのような
      タイプがありますが、同様に保証外とさせていただきます)。

物理演算について

    * 必要最低限と思われる剛体のみ実装しています。
      
モーフについて

    * 複数のモーフで表情が完成するものもありますが、逆に破綻するものもあり
      ます。ブレンドの合計値が 100%を超えないあたりで使用してもらえれば
      大丈夫かと(それでも破綻する場合はありますが)。
その他
  
    * このタイプのモデルは基本的に可能な限りシンプルな構造にしようと考えて
      います。逆に言えば、使う人のニーズで改造しやすいと思います。

    * エプロンは透過を使用してます。モデルへのカメラの視点方向とその背景物
      の描画順序によっては表示が乱れる可能性がありますので、適時MMDの描画順
      で調整してください。
